# Poverty Risk Prediction Model

A Logistic Regression model to predict poverty-risk households using the UCI Adult Census Income dataset.

## Dataset

Uses the Adult Census Income dataset from UCI Machine Learning Repository containing 30,000+ records with demographic and employment features.

## Features

- Age
- Work class
- Education level
- Marital status
- Occupation
- Relationship
- Race
- Sex
- Capital gain/loss
- Hours per week

## Requirements

```
pip install scikit-learn pandas numpy
```

## Usage

```
python main.py
```

## Output

- Model accuracy and classification metrics
- Feature importance coefficients
- Sample predictions from test set
